## How to Use

Skills are loaded by AI agents based on the task context. Each skill file contains:

- YAML frontmatter with metadata for discovery
- Implementation patterns and best practices
- Code examples and common pitfalls
- Security considerations where applicable

## Contributing

1. Create a new folder under the appropriate language directory
2. Add `SKILL.md` with proper YAML frontmatter
3. Include `LICENSE.md` in the language root
4. Update the relevant `README.md` file
